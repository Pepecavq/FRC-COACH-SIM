# FRC-COACH-SIM
Un programa creado para entrenar a Coaches de robótica y facilitar la comunicación efectivca entre coaches y drivers
La idea de este proyecto fue comenzada gracias a nuestro compartido gusto por la robótica deportiva, 3 de nosotros hemos estado en robótica durante nuestra etapa de prepa, y hemos notado problemas en muchas partes, uno de ellos, la falta de formas de entrenamiento para drive coaches.

Gracias a este problema hemos desarrollado un programa con ayuda de Gemini API y el concept robot de Wcp 2026 para hacer una simulación entre coach (Jugador) y driver (IA).

El concepto es simple, ayudar a controlar la presión, y ayudar a que den instrucciones más claras para facilitar una mejor comunicación entre driver y coach.

El simulador consiste en una simulación de juego de la temporada de FRC 2026 “Rebuilt” donde la IA maneja el robot siguiendo las instrucciones del Coach, también se adapta a cambios de tono, y simula el estrés.

La ayuda externa, como mentores y opiniones con experiencia, nos ayudaron a desarrollar nuevas ideas y personalizarlo a gusto de gente con experiencia real.



Nuestro proceso creativo fue lento pero eficiente, tuvimos una idea y la empezamos, durante su creación notamos fallas y opiniones que contrastan, por ese motivo lo fuimos modificando, el proceso creativo nunca acabó hasta el final de la creación del programa lo que nos permitió siempre innovar y dar el mejor producto posible.

Finalmente cabe recalcar que este proyecto se va a subir a Github para hacerlo alcanzable para cualquier driver o coach, y mejorar la experiencia de la robótica para todos.
